# Plan: Create Vect Theme for POS System
**Date**: 2025-11-15
**Objective**: Create a new minimalist "Vect" theme based on pos-3.jpg design

## Problem Analysis
- Current system has 3 themes: Evolution, Restaurant, Oblivion
- Need to add a fourth theme called "Vect" based on the clean, minimal Odoo-style design from pos-3.jpg
- Design should follow modern 2025 UI/UX trends with "Bold Minimalism" approach

## Design Requirements (Based on pos-3.jpg Analysis)
### Visual Characteristics:
- **Background**: Pure white (#ffffff)
- **Layout**: Clean grid-based layout with clear sections
- **Typography**: Modern, sans-serif font (Inter or similar)
- **Color Palette**: 
  - Primary: White (#ffffff)
  - Secondary: Light gray (#f5f5f5)
  - Borders: #e1e1e1
  - Text: Dark gray (#333333)
  - Accent: Teal/cyan (#20B2AA) for primary actions
- **Components**: 
  - Product grid with image cards
  - Left sidebar with cart and numpad
  - Top navigation with search
  - Clean button styles with subtle shadows

### Layout Structure:
1. **Header**: Simple navigation bar with search functionality
2. **Main Area**: Product grid with category tabs
3. **Sidebar**: Cart summary and digital numpad
4. **Theme Switcher**: Integration with existing theme system

## Technical Architecture
### Files to Create:
